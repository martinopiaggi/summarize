# Summarize

Transcribe and summarize videos from YouTube, Instagram, TikTok, Twitter/X, Reddit, Facebook, Google Drive, Dropbox, and local files. Works with any OpenAI-compatible LLM provider, including locally hosted endpoints.

## Requirements

- **Python** 3.7+
- **ffmpeg** (must be on `PATH`)
- At least one **LLM API key** in `.env`
- **Cobalt** fallback downloader for URLs that yt-dlp cannot handle (included in Docker Compose). See [Cobalt](cobalt).

## Quick Start

```bash
git clone https://github.com/martinopiaggi/summarize.git
cd summarize
pip install -e .
python -m summarizer --source "https://youtube.com/watch?v=VIDEO_ID"
```

The summary is saved to `summaries/watch_YYYYMMDD_HHMMSS.md`.

## Pick Your Interface

| Interface | Command |
|-----------|---------|
| **CLI** | `python -m summarizer --source <source>` |
| **HTTP API** | `python -m summarizer serve` → `http://localhost:8000/docs` |
| **Streamlit GUI** | `python -m streamlit run app.py` |
| **Docker** | `docker compose up -d` → `http://localhost:8501` |
| **Agent Skill** | [`.agent/skills/summarize/SKILL.md`](https://github.com/martinopiaggi/summarize/blob/main/.agent/skills/summarize/SKILL.md) |

See the [HTTP API guide](api-reference/http-api) for endpoint details, request examples, and CORS configuration.
